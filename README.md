# SunTrack — Sitio web (Entrega final)

Este repositorio contiene la estructura y archivos para el sitio web del proyecto SunTrack.

## Estructura
- `index.html` — Página principal (Home)
- `quienes.html` — Quiénes somos (Misión / Visión / Valores)
- `catalogo.html` — Catálogo de productos
- `contacto.html` — Contacto / Formulario de compra
- `proyecto.html` — Documento del proyecto (resumen, links a Google Docs/Trello/Git)
- `styles.css`, `script.js` — Recursos comunes
- `suntrack.png` — Imagen del producto (poner archivo en la misma carpeta)

## Instrucciones para el docente
1. Revisar `proyecto.html` para un resumen del proyecto y links a la documentación online.
2. Los documentos principales (EDT, GANTT, FODA) deben subirse a Google Docs y enlazarse en `proyecto.html`.

## Cómo probar localmente
Abrir `index.html` en un navegador moderno (Chrome, Edge, Firefox). Para probar el sitio con un servidor local (recomendado):

```powershell
# en Windows PowerShell, dentro de la carpeta del proyecto
# si tiene Python 3 instalado:
python -m http.server 8000
# luego abrir http://localhost:8000/index.html
```
